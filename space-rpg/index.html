QUEST STEPS

Add a lootable item to the destination. 
<place-item dest="Backwater" item="Shiny Meteorite" />
Add a purchasable item to the destination's trader. 
<place-inventory dest="Backwater" item="Phased Ray" />
Display a message. If the optional ok and cancel attributes are provided they will appear as buttons. Cancel will rewinds the quest one step, so that the same trigger that triggered the quest will display the same message the next turn. Ok will progress the quest as normally. 
<dialog>"Hey hitchhiker!"</dialog>
<dialog ok="YES PLEASE" cancel="NO THANKS">"Would you like some tea?"</dialog>
Include this quest in the quest list. 
<add-to-quest-list />
Pause the quest until all conditions are fulfilled. All attributes are optional. the trigger attribute take either "auto" or "action" and control if the condition must be triggered with the TRIGGER button on the radar screen or not. 
<condition
      have-item="Shiny Meteorite"
      at-dest="Backwater"
      not-at-dest="Saturn"
      have-credits="3000"
      trigger="auto" />
Add items or credits to the player's inventory. 
<give credits="900" />
<give item="Phased Ray" />
Remove an item or credits from the player's inventory. 
<take credits="900" />
<take item="Phased Ray" />
Place a quest marker on the map. color and icon attributes are optional. icon can take these values: "arrow", "square", "enter", "exit". A quest can only have a single map marker. 
<mark-on-map dest="Samson's Folly" color="0x0F0" icon="arrow" />
Change the player's ship. 
<change-ship ship="clipper" />
Clear the map marker for this quest. 
<remove-map-marker />
Transports the player to another destination. 
<travel dest="Secret Station" />
Rewinds or fast forwards the quest a set number of steps. 
<goto jump="-3" />
